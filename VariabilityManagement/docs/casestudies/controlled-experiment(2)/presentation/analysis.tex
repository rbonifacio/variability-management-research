\section{Background}

\begin{frame}
\frametitle{Objetivos do experimento}

Comparar duas abordagens propostas para gerencias variações 
em requisitos de software: PLUSS e MSVCM. 

\begin{block}{Hipóteses}
MSVCM...
\begin{enumerate}
\item reduz o espalhamento de features
\item aumenta a coesão dos cenários
\item suporta a evolução de acordo com o princípio \emph{open-closed}
\item não impacta o tempo para extrair ou evoluir LPS

\end{enumerate}

\end{block}

\end{frame}

\begin{frame}
\frametitle{Linhas de produtos de software}

Abordagem sistemática de reúso, em que programas pertencentes 
a mesma família são gerados a partir da seleção e configuração 
de artefatos reusáveis. Um membro da família é caracterizado por meio de 
uma escolha válida das features que caracterizam a linha de produtos.

\end{frame}

\section{Segunda iteração}

\begin{frame}
\frametitle{Visão geral}

\begin{itemize}
\item Semelhante a primeira iteração, investigamos
tanto a abordagem extrativa quanto evolutiva de 
linhas de produtos. 
\item Diferenças:
\begin{itemize}
\item Mais alunos envolvidos nos experimentos
\item Mais aulas preparatórios para as atividades 
\item Uso de uma ferramenta para especificar e coletar o tempo
\end{itemize}

\end{itemize}


\end{frame}

\begin{frame}
\frametitle{Análise de variância - Extração de LPS}
Dado o modelo:
\begin{small}
\center{
$Response = Replic+Student:Replic+Student+SPL+Technique$
}
\end{small}

\begin{enumerate}
\item Checar as seguintes propriedades:
\begin{itemize}
\item modelo apresenta variância constante dos erros 
\item modelo é aditivo
\item modelo apresenta distribuição normal dos resíduos
\end{itemize}
\item Verificar se o fator técnica é influente (p-test)
\end{enumerate}

\end{frame}

\begin{frame}
\frametitle{Ajuste do modelo}
Técnica de Box Cox
\center{
\includegraphics[scale=0.3]{images/boxcox.eps}
}
\begin{itemize}
\item Sem necessidades de ajuste no modelo
\end{itemize}
\end{frame}

\begin{frame}
\frametitle{Variância constante dos erros}
Resíduos x Valores Esperados
\begin{center}
\includegraphics[scale=0.3]{images/residuos.eps}
\end{center}
\end{frame}

\begin{frame}
\frametitle{Variância constante dos erros}
\emph{Score Test for Non-Constant Error Variance}
\begin{itemize}
\item $H_0$: variância constante dos erros
\item $H_1$: $H_0$ é falsa 
\end{itemize}
\begin{center}
($p_{value} = 0.62) \Rightarrow $ Sem evidências para rejeitarmos $H_0$
\end{center}
\end{frame}

\begin{frame}
\frametitle{Modelo é aditivo}
Teste de não aditividade de Tukey
\begin{itemize}
\item $H_0$: o modelo é aditivo
\item $H_1$: $H_0$ é falsa 
\end{itemize}

\begin{center}
($p_{value} = 0.50) \Rightarrow $ Sem evidências para rejeitarmos $H_0$
\end{center}

\end{frame}
